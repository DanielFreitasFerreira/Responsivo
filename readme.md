## CSS Units

Unidades de medidas do CSS

Layout Fixo

`px` - Pixels

O valor definido em (px) vai sempre o mesmo independente da resolução / tamanho da tela.

Layout Fluido

`%` - Porcentagem

`auto` - Automático

`vh` - Viewport Height

`vw` - Viewport Width
